Profile Editor is a handy plugin, with a simple interface, for managing user profile.

== Description ==

Profile Editor allows you to create front-end login, user registration and edit profile pages by using shortcodes.

== Installation ==

Installing "Profile editor" can be done either by searching for "Profile editor" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
